Create a modern, premium 3D website for a vegetarian Italian cafe brand named "Meloghrano Cafe".

The design should feel aesthetic, cozy, and vibrant — inspired by pastel Italian cafes with a premium touch. The goal is to impress customers and increase reservations.

### Core Requirements:

* Fully responsive (mobile-first design, works perfectly on phone + desktop)
* Smooth animations and 3D effects using modern libraries (Three.js / GSAP / Framer Motion)
* Fast loading and clean UI
* Soft pastel + warm color palette (peach, cream, sage green, soft red)

---

* 3D rotating gelato or pizza model

  * "View Menu"
  * "Reserve Table"
